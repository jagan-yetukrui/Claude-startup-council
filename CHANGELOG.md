# Changelog

## 1.0.0 — first public release

The council as a distributable Claude Code skill.

- **Three research scouts** build one evidence-graded dossier (product, disconfirming voices +
  graveyard, market/analog structure).
- **Three members** — Believer, Skeptic, Investor — each file an independent research report.
- **Live courtroom hearing (Round 2):** the members argue turn by turn, objecting to each other's
  specific points, until the weak findings die.
- **A Judge** weighs both the reports and the hearing, applies a domain-specific lens, and delivers
  one verdict in plain, human language — no jargon, no AI-slop.
- **Honest in both directions:** a confident BUILD when the evidence earns it, a KILL only when the
  evidence earns that. It doesn't try to prove you right or wrong.
- **Early-stage rule:** never asks for revenue, sales, LOIs, or prepays; reasons from market research.
- **Deterministic scoring and ledger** via bundled Python (`scripts/score.py`, `scripts/ledger.py`).
- **Append-only ledger** with retests and a self-calibration record. Your ledger stays local and is
  never published.
